# 🌌 AXIOM — AI Ethics & Responsible AI Oracle

AXIOM is a futuristic AI chatbot that provides accurate answers to questions related to **AI Ethics, Fairness, Transparency, Privacy, Governance, and Responsible AI**. Built using **Python, NLTK, Scikit-Learn, and Tkinter**, it combines NLP techniques with an immersive cyberpunk-inspired interface.

## ✨ Features

- 💬 AI Ethics Q&A with 25+ curated expert FAQs
- 🧠 NLP-powered query understanding
- 🎯 TF-IDF + Cosine Similarity matching
- 📊 Confidence score visualization
- 🌌 Animated Nebula-themed UI
- ⚡ Fast, lightweight, and fully offline
- 📚 Covers Bias, Fairness, XAI, Privacy, Deepfakes, AI Laws, Governance, and more

## 🛠 Tech Stack

- **Python**
- **Tkinter**
- **NLTK**
- **Scikit-Learn**
- **NumPy**

## 📂 Project Structure

```text
axiom.py            # Main GUI application
chatbot_engine.py   # NLP retrieval engine
faq_data.py         # AI Ethics knowledge base
requirements.txt    # Dependencies
```

## ⚙️ How It Works

```text
User Question
      ↓
NLP Preprocessing
      ↓
TF-IDF Vectorization
      ↓
Cosine Similarity Matching
      ↓
Best FAQ Retrieved
      ↓
Answer + Confidence Score
```

## 🚀 Installation

```bash
git clone <repository-url>
cd AXIOM
pip install -r requirements.txt
python axiom.py
```
